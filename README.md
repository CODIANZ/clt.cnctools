## 注意 : macOS での python 2

macOS 12.3 (Monterey) から システム の python 2 が削除となっています。 <br>
この プロジェクト では インストール時 に python 2 が必要となっているため、以下を参考に python 2 環境を準備する必要があります。 <br>

https://www.python.jp/install/macos/index.html
